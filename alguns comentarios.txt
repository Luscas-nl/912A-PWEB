---------- HTML ----------

## OBJETOS ##
<title> = Titulo da pagina
<link> = Usado para importar algo para o arquivo html
<a> = Link
<div> = "Caixa"
<h1> = Titulo
<p> = Paragrafo

## DEFINIÇÕES ##
class = classe do objeto, a qual vai ser usada como referencia dele no arquivo css para estilização
href = referencia de pra onde o link vai redirecinar ao ser clicado ou no caso do <link> ele funciona como o [src] ↓
src = fonte/arquivo a ser usado
rel = oq a importação vai ser [stylesheet = folha de estilo ou no caso um arrquivo pra estilização (css)] [shotcut icon = icone da pagina]
alt = titulo da imagem caso ela nn carregue, e serve como referencia dela pro navegador tbm


---------- CSS ----------

## @ ##
@import: é pra importar algo, nesse caso usamos uma url pra importar as fontes do projeto, só importamos fontes mesmo
@keyframe: usado para criar uma animação
 - from: como o objeto vai estar no inicio da animação
 - to: como o objeto vai estar no final da animação
@media: tambem chamado de media query, usado pra fazer modificações na estilização de um objeto caso uma condição seja verdadeira. nesse caso usado para analisar o tamanho da tela e mudar as estilizações para telas menores

## PSEUDO-ELEMNTOS ##
::after = cria um pseudo-elemento que se tornará primeiro filho do elemento referenciado
::before = faz a mesma coisa que o after
:hover = basicamente especifica uma estilização para o elemento referenciado quando o mouse passar em cima dele

O resto das coisas fazem parte do proprio css, no caso as estilizações, para referenciar o objeto o qual queremos fazer 
a estilização utilizamos (.) e o nome que foi dado na classe do elemento; Exemplo:
    .boxPrincipal{}

e dentro das {} colocamos a estilização do elemento
